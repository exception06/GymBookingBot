# Gym Booking Bot 

Automates booking of Tuesday and Thursday 6:00 PM classes from [AppBrewery Gym](https://appbrewery.github.io/gym/) using Selenium and a persistent Chrome profile.

## Features
- Auto-login with saved credentials
- Filters classes by day and time
- Books or waitlists based on availability
- Verifies bookings on the confirmation page

## Setup

### Requirements
- Python 3.8+
- Google Chrome
- ChromeDriver (matching your Chrome version)
